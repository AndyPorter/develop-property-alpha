Ready to sell properties

I'm a developer and I'm ready to sell properties...

These prototypes show how the developer could indicate their readyness to sell - exchange, complete, transact, etc - to the buyer

We think that it is useful to the developer to be able to see if there would be any problems registering this property/title before the sale/purchase transaction takes place.

We think that it is useful to the purchase - buyers conveyancer (and others?) to be able to see if there would be any problems registering this property/title before the sale/purchase transaction takes place.

Prototypes

  • Version 2: I'm a developer and a want to indicate i'm ready to sell (there is nothing else i expect to do to this property/plot)
  • Version 1: I'm a developer and a want to 'lock the plot' (there is nothing else i expect to do to this property/plot)

Alternative methods

There are other ways we could achieve these outcomes. These prototypes are to stimulate conversation with conveyancers as to the concept and problems experienced in sales/purchases.
